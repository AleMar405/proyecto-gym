# proyecto-gym

----Descripcion del proyecto----
Este proyecto tiene cómo objetivo una plataforma web dónde:
-Se puedan registrar e iniciar sesión entrenadores y usuarios.
-Crear y administrar rutinas de entrenamiento.
-Asignar rutinas a usuarios.
-Registrar el progreso de entrenamiento de cada usuario.


----Tecnologias----
-MongoDB
-Git Hub
-Node.js
-React
-Express.js

---Cómo ejecutar el proyecto en un entorno local---
1. Copiar la url de la dirección de donde se desea crear la carpeta.

2. En el cmd poner: cd (ruta) y dar enter.


3. Copiar la ruta del repositorio.

4. En el cmd poner: git clone (ruta) y dar enter.


----Estructura del proyecto----

Este proyecto esta dividido en dos carpetas:

Backend/
-Base de datos 

fronted/ 
-Interfaz de usuario
